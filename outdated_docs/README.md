# Устаревшие материалы

В этот каталог перемещены документы, которые больше не используются в проекте и требуют удаления либо полной переработки. Список и краткое описание:

- `tmp_base_doc.html` и `tmp_base_doc.txt` — выгрузка публичной документации Base dVRF без привязки к Supra Lottery.
- `migration_dvrf3.html` и `migration_dvrf3.txt` — сохранённая страница миграции Supra dVRF v3, дублирующая официальный сайт.
- `tmp_crystara_readme.md` — копия README стороннего проекта Crystara SDK.
- `audit/` — дубликат отчётов из `SupraLottery/docs/audit`, поддерживать обе копии нет смысла.
- `supra_labs_reference_analysis.md` — временный отчёт по сравнению с Supra-Labs, который более не отражает актуальное состояние.
- `fungible_asset_transition_plan.md` — черновик перехода на Supra Fungible Asset; фактическая реализация выполнена и документ более не обновляется.
- `supra_progress.md` — отчёт о ходе интеграции Supra, устаревший после завершения перечисленных задач.
- `move_common_errors_ru.md` — локализованный справочник ошибок Move, который больше не поддерживается и дублирует обновлённые инструкции в основном репозитории.
- `supra-move-mcp/` — клон внешнего репозитория без исходников Move, использовался временно для экспериментов с MCP и более не нужен.
- `SUPRA_FIX_PLAN.md` — план устранения замечаний Supra, закрытый после выполнения всех пунктов.
- `alignment_gaps.md` — список расхождений с требованиями Supra, актуализированный и перенесённый в другие документы.
- `visibility_risk_assessment.md` — оценка рисков видимости, перекрытая финальными отчётами аудита.
- `supra_alignment_plan.md` и `supra_alignment_status.md` — дорожная карта и статус выравнивания, завершённые после миграции на Supra dVRF v3.
- `move_migration_next_steps.md`, `move_migration_execution_plan.md` и `move_migration_compilation_fix_plan.md` — планы миграции Move, реализованные в рамках релиза SupraLottery.

До принятия решения об окончательном удалении файлы находятся здесь, чтобы не загромождать основную документацию репозитория.
